# Guía de Copywriting Publicitario

## Fórmulas de Copywriting

### AIDA (Atención - Interés - Deseo - Acción)
1. **Atención:** Captura con un dato impactante o pregunta provocadora
2. **Interés:** Desarrolla el problema o beneficio
3. **Deseo:** Muestra la transformación o resultado
4. **Acción:** Llamada a la acción clara y directa

### PAS (Problema - Agitación - Solución)
1. **Problema:** Identifica el dolor del cliente
2. **Agitación:** Amplifica las consecuencias de no actuar
3. **Solución:** Presenta el producto/servicio como la respuesta

### BAB (Before - After - Bridge)
1. **Before:** Situación actual del cliente
2. **After:** Cómo será después de usar el producto
3. **Bridge:** El producto/servicio es el puente

## Tips por Industria

### Tecnología
- Enfatizar innovación y eficiencia
- Usar datos y métricas cuando sea posible
- Hablar de transformación y futuro
- Palabras clave: automatizar, optimizar, escalar, inteligente

### Gastronomía / Restaurantes
- Apelar a los sentidos (sabor, aroma, textura)
- Crear urgencia con ofertas limitadas
- Destacar ingredientes y calidad
- Palabras clave: fresco, artesanal, irresistible, sabor auténtico

### Moda / Retail
- Apelar a la identidad y estilo de vida
- Crear exclusividad y tendencia
- Usar storytelling emocional
- Palabras clave: exclusivo, tendencia, estilo, único

### Salud / Bienestar
- Generar confianza con datos y testimonios
- Mostrar resultados reales
- Tono empático y profesional
- Palabras clave: bienestar, transformación, cuidado, vitalidad

### Educación
- Destacar oportunidades y crecimiento
- Mostrar casos de éxito
- Apelar a la superación personal
- Palabras clave: crecer, aprender, futuro, oportunidad

### Finanzas / Seguros
- Transmitir seguridad y confianza
- Usar datos concretos
- Simplificar conceptos complejos
- Palabras clave: proteger, crecer, seguridad, tranquilidad

## Reglas Generales
- Frases de máximo 15 palabras para impacto
- Usar verbos de acción en imperativo
- Incluir beneficio claro para el cliente
- Evitar tecnicismos innecesarios
- Adaptar el tono a cada marca
- Generar emoción y conexión
